# 🔲 Python QR Code Generator

A simple QR Code Generator built with Python that converts 
any URL or text into a scannable QR Code image.

## 🛠️ Libraries Used
- Python 3
- qrcode
- Pillow (PIL)

## ⚙️ Features
- Custom colors (fill & background)
- High error correction (works even if 30% damaged)
- Saves as PNG image
- Works with any URL or text

## 🚀 How to Run

# Clone the repo
git clone https://github.com/your-username/qr-code-generator

# Create virtual environment
python -m venv venv
source venv/bin/activate

# Install libraries
pip install "qrcode[pil]" pillow

# Run
python qr_code.py

## 📤 Output
Generates a high quality PNG QR Code image!
